# OrgoMastery (Frontend)

## Overview

**OrgoMastery** is a modern educational web application designed to help students master Organic Chemistry through structured lectures, interactive quizzes, and organized learning resources.

The platform presents complex chemistry topics through guided lessons that include lecture videos, notes, and quizzes. The goal is to make Organic Chemistry more approachable by presenting concepts through patterns, practice, and structured learning paths.

This project combines my experience as a **college organic chemistry professor** with my ongoing transition into **software engineering**, applying modern front-end technologies to build intuitive educational tools.

---

## Purpose

OrgoMastery was built to address common challenges in chemistry education by:

- Organizing complex topics into structured lessons
- Supporting active learning through quizzes and feedback
- Providing a clean, distraction-free learning interface
- Demonstrating how modern web technologies can improve educational tools

The application also serves as a **portfolio project** showcasing my skills in:

- React development
- Component-based architecture
- Responsive UI design
- Integrating front-end applications with backend APIs

## Features

### Lecture Library
- Topic-based organization of Organic Chemistry material
- Dedicated lecture pages containing videos, notes, and quizzes
- Structured learning progression

### Interactive Quizzes
- Multiple-choice questions tied to lecture content
- Immediate feedback and score calculation
- Quiz attempt tracking

### Progress Tracking
- Students can monitor quiz attempts and scores
- Provides insight into learning progress over time

### Instructor Tools
Instructors can create and manage:

- Lectures
- Videos
- Lecture notes
- Quizzes
- Quiz questions

### Discussion & Comments
- Students can engage with content through comments
- Encourages discussion and reflection

### Responsive Design
- Optimized for desktop, tablet, and mobile devices
- Layout built with responsive CSS

## Application Structure

Key frontend components include:

- `Home` – Landing page introducing the platform  
- `Header` – Navigation and authentication controls  
- `LectureList` – Displays available lectures  
- `LecturePage` – Displays lecture videos, notes, and quizzes  
- `QuizPage` – Interactive quiz interface  
- `MyProgress` – Displays quiz attempts and student performance  
- `InstructorDashboard` – Instructor management interface  
- `InstructorLectureManager` – Manage lectures, videos, notes, and quizzes  
- `Contact` – User feedback and communication page  
- `About` – Overview of the platform and its mission

## Design Philosophy

OrgoMastery prioritizes:

- Clear navigation and logical information flow
- Minimal visual clutter to reduce cognitive load
- Consistent layout and typography
- Educational clarity over visual complexity

Many design decisions were informed by real classroom teaching experience and common student challenges in Organic Chemistry.
